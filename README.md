# Sales Performance Analysis using Python

## Objective
Analyze sales data to identify trends, regional performance, and top products.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset contains sales transactions with order date, region, product, sales amount, and profit.

## Key Analysis
- Monthly sales trend
- Region-wise performance
- Top 5 products by sales
- Profit analysis

## Key Insights
- North region generated the highest revenue
- Electronics category contributed maximum sales
- Sales increased steadily over time

## How to Run
1. Clone the repository
2. Open the notebook in Jupyter
3. Run all cells
